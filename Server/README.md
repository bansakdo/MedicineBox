### 자동 약 제공 및 의약품 관리 시스템 프로젝트 입니다.

사용자가 안드로이드 애플리케이션을 통해 복용해야 하는 약을 정해진 시간에 구급함에서 제공받는 시스템 입니다.

해당 프로젝트는 관리자용 웹, 안드로이드 앱, 스마트 구급함(디바이스), 서버 총 4개의 영역으로 구성되어 있으며, 해당 레파지토리는 서버 레파지토리 입니다.
